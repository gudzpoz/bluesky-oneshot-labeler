Oneshot Labeler

Labeler

This website hosts a simple Bluesky labeler available at @{{.User}}.

What It Does

Basically, it reads the labels from @{{.Upstream}}, which mostly does post-wise moderation. For every bad post, it marks the user of that post as a bad actor and labels them as such on a user-wise basis. Check out {{.User}} for more info.

Feed

We also provides a feed that excludes posts from said users (plus a curated block list). You (probably) can find it at {{.User}}/feed/oneshot.

Source Code

The source code is available at gudzpoz/bluesky-oneshot-labeler.
